# A.W.A.R.E.
The A.W.A.R.E. system acts as a centralized platform that bridges the gap between quantitative academic grades, qualitative student well-being, and personal study habits. By changing from massive, one-time surveys to anonymous, weekly "Pulse Checks," the system can obtain accurate, honest class reviews.

How to use this demo version:
1. Run Apache and MySQL on XAMPP
2. Run server.py
3. Run main.py (ignore dashboard.py for now)

Notes:
- Data are currently hardcoded
- Evaluation from student not updating properly on admins' end
